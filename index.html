Профессиональный профиль

Я обучаюсь по специальности "Компьютерный дизайн" — современному направлению, объединяющему художественное творчество с цифровыми технологиями. В рамках обучения я осваиваю основы композиции, цветоведения и типографики, работу с графическими редакторами (Adobe Photoshop, Illustrator, Figma), создание пользовательских интерфейсов (UI/UX), 3D-моделирование и визуализацию, а также анимацию и создание интерактивных элементов.

Параллельно я активно изучаю фронтенд-разработку, что позволяет мне не только проектировать интерфейсы, но и реализовывать их в коде. Мои технические навыки включают: HTML5 (семантическая разметка, доступность), CSS3 (Flexbox, Grid, адаптивный дизайн), препроцессоры SASS/SCSS для эффективной стилизации, CSS-анимации, JavaScript (ES6+), DOM-манипуляции для создания динамических интерфейсов, а также изучение React для компонентного подхода.
